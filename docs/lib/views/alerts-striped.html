Colorstrap Striped Alerts

Did you know you could apply the striped effect to Bootstrap alerts? Well, you can! And, you can also combine with with all 140 HTML named colors!

Example

Brown Striped Alert! You successfully read this important Brown alert message
<div class="alert alert-brown colorstrap-striped"><strong>Brown Striped Alert!</strong> You successfully read this important Brown alert message</div>
{{key}} Striped Alert! You successfully read this important {{key}} alert message
