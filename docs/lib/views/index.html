Go beyond Default, Info, Success, Warning and Danger with new Bootstrap-ready color classes like HotPink, MidnightBlue, DarkSeaGreen and any one of the 140 named HTML colors!


Download Colorstrap

Colorstrap is currently compatible with Bootstrap version 3.

You like colors, right?

Use Colorstrap classes with Bootstrap components including Labels, Buttons, Jumbotrons, and even Modals!

And there's more than colors!

Colorstrap also brings stripes, themes, and outlines to many of Bootstrap's components.

Colorstrap is Open Source.

You can find the Colorstrap source on Github. Browse, Fork or raise an issue.
