2. Options for use should be one giant css file, a minified css, a css for each type of bootstrap object (colorstrap-bg.min.css)

3. The background is always the reference color. So:
	.alert-beige {
		background-color: #F5F5DC;
		...
	}
4. The text color 15% darker than the background
5. The border color is 5% darker than the background

Examples:

	.alert-burlywood {
		backgroud-color: #DEB887;
		color: #BC9C72;
		border-color: #D2AE80;
	}
